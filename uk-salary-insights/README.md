# 💸 UK Salary Insights 2024

An interactive dashboard exploring salaries across jobs, cities and industries in the UK.

## 🔗 Live Dashboard
👉 [Click here to view]()

## 📊 Features
- 🗺️ Interactive UK salary map by city
- 💼 Top 20 highest paying jobs chart
- 🏙️ City salary comparison with heatmap
- 📈 Salary progression by experience level
- 🏭 Industry salary breakdown
- ☀️ Sunburst chart by industry and job
- 🎻 Violin and box plots
- 🔍 Search any job title for detailed insights
- 🎨 Beautiful dark gold theme

## 🛠️ Built With
- Python
- Pandas
- Plotly
- Streamlit
- NumPy

## 🚀 How to Run Locally
1. Clone this repo
2. Install requirements:
```
   pip install -r requirements.txt
```
3. Run the app:
```
   streamlit run app.py
```
