# 💰 Personal Finance Tracker

A beautiful and interactive personal finance tracker built with Python and Streamlit.

## 🔗 Live App
👉 [Click here to view](https://finance-tracker-ylheucbkc3pn9itzmwszea.streamlit.app)

## 📊 Features
- 💚 Track income by category
- 🔴 Monitor expenses by category
- 📈 Monthly trend analysis
- 📊 Income vs Expenses charts
- 📋 Full transaction history
- 💾 Data saves automatically
- 🎨 Beautiful dark theme

## 🛠️ Built With
- Python
- Pandas
- Plotly
- Streamlit

## 🚀 How to Run Locally
1. Clone this repo
2. Install requirements:
```
   pip install -r requirements.txt
```
3. Run the app:
```
   streamlit run app.py
```
