# 🚔 UK Crime Dashboard

An interactive real-time UK Crime Data Explorer built with Python and Streamlit.

## 🔗 Live Dashboard
👉 [Click here to view](https://uk-crime-dashboard-dzbwmzdq4ttxyi77gwfvrm.streamlit.app)

## 📊 Features
- 🗺️ Interactive crime map of UK cities
- 🔽 Filter by city and crime type
- 📊 Crime category bar charts
- 🥧 Outcome distribution pie chart
- Real data from UK Police API

## 🛠️ Built With
- Python
- Pandas
- Plotly
- Streamlit
- UK Police API (data.police.uk)

## 🚀 How to Run Locally
1. Clone this repo
2. Install requirements:
```
   pip install -r requirements.txt
```
3. Run the app:
```
   streamlit run app.py
```
